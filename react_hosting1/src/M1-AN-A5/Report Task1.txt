Purpose of the document
This document explains the various activities performed as part of the Testing of the ‘Product Display’ react application


Application Overview
Product Display is a React App which is used to display product details in tabular form with other Functionalities.
Testing Scope
1. In Scope
2. Out of Scope
In-Scope  : Functional Testing for the component are in Scope of Testing
Out of Scope: Performance Testing was not done for this application
Functionalities
Product Display renders product details in tabular form display details like product name total sales,etc .Product display have 3 columns which are used to display  Product name ,Total Sales and last one is used for Details button for more details on sales of product in form of list in every row of table.In Product Display on top we have three buttons use to sort the table by product , total sale in ascending or descending order.
