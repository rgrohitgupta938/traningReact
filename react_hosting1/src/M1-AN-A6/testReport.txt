Purpose of the document
This document explains the various activities performed as part of the Testing of the ‘Football tournament’ react application


Application Overview
Football tournament is a React App which is used to display product details in tabular form with other Functionalities.
Testing Scope

2. Out of Scope
In-Scope  : Functional Testing for the component are in Scope of Testing
